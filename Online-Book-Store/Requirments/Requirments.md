# Online Book Store - Requirements
## Project
Online Book Store
## Feature
 Login

### Functional Requirements

FR-001: The user can log in using a valid email and password.

FR-002: The email must be in a valid email format.

FR-003: The password must contain at least 8 characters.

FR-004: If the email or password is incorrect, the system should display an error message.

FR-005: If the email or password field is empty, the system should display a validation message.

FR-006: After a successful login, the user should be redirected to the Home page.
